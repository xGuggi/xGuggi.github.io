Banking App

This project is a web application designed for banking or financial management purposes. It includes the following features:

  • Authentication System: Users can securely log in using their username and password. Session management ensures users stay logged in across multiple pages.
  • User Account Management: Upon login, users can view their account information, including balances for different account types such as checking, savings, and high yield.
  • Transaction Handling: Users can perform financial transactions like withdrawals, deposits, and transfers between accounts. Input fields allow users to specify transaction amounts and select accounts.
  • Transaction History: The application provides users with the ability to review their transaction history, including options to view history for specific accounts or for all accounts.
  • User Interface: Designed with React.js, the interface is dynamic and responsive, featuring modal components for additional transaction details and history. Form validation and error handling enhance the user experience.
  • Backend Server: The frontend communicates with a backend server built with Node.js, Express.js, and MongoDB. The backend handles authentication, user account management, transaction processing, and data retrieval.

Overall, this project aims to provide users with a secure and user-friendly platform for managing their finances online.
